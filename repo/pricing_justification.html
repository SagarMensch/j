2. Cloud Infrastructure Sizing (AWS Data Center)

The system is physically hosted in the heavily secured AWS Mumbai Region. The infrastructure costs are highly optimized utilizing AWS Reserved Instances.

  • App Server (2x `t3.xlarge`): Handles 30 concurrent users effortlessly. Cost: ~₹20k/mo.
  • AI/ML Processing (2x `g4dn.xlarge` via AWS): Because you are utilizing a custom Fine-Tuned Mistral model, AWS provides the NVIDIA GPUs on rent natively. You do NOT need to buy them. AWS provisions these automatically to ensure rapid AI speeds. Cost: ~₹65k/mo.
  • Database (PostgreSQL RDS): Utilizing pgvector natively within AWS RDS. By consolidating user data and AI vectors into Postgres, you avoid paying massive premiums to 3rd-party vector tools like Pinecone or Qdrant. Massive cost saving.

4. On-Premise vs Cloud Deployment Rules

Hardware Rule: If they elect to go 100% On-Premise (Local Air-Gapped Data Center instead of AWS): They avoid the Annual AWS Cloud Infrastructure costs entirely. However, they must physically purchase the Server Hardware and NVIDIA GPUs for their server room, which becomes their internal CapEx cost.
  • AWS Cloud (Default): ~₹8.5 Lakhs/yr. You rent everything (including GPUs for Mistral) from AWS. Zero hardware purchase required.
  • On-Premise (Strict Security): They pay you an additional ₹8 Lakhs One-Time Setup to install it in their physical datacenter. They must provide the server hardware.
